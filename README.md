Sistema de Bate-Papo em C#

Projeto desenvolvido para estudo de comunicação em rede utilizando C# e .NET.

A proposta é construir um sistema de chat passo a passo, entendendo o funcionamento interno dos sockets antes de utilizar bibliotecas de alto nível.
- Comunicação básica

Implementação de conexão TCP entre um cliente e um servidor rodando localmente.

Funcionalidades atuais:

Servidor escutando conexões

Cliente conectando via localhost

Troca de mensagens em tempo real

Comunicação bidirecional

Uso de threads para envio e recebimento simultâneo

- Próximas etapas

Múltiplos clientes simultâneos

Broadcast de mensagens

Sistema de nomes de usuário

Interface gráfica

- Tecnologias utilizadas

C#

.NET

TCP Sockets

Multithreading
